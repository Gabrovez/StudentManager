# StudentManager

##назначение программы
Приложение для управления списком студентов с использованием базы данных SQLite. Позваоляет добавлять, просматривать и удалять записи о студентах.

##Реализованные функции
- Подключение к локальной базе данных SQLite
- Отображение списка студентов в таблице
- Добавление новых стундентов с валидацией ввода
- Удаление студентов с подтвержений действия
- Защита от SQL-инъекций с ипсользованием параметризованных запросов
- Обработка исключений для повышения стабильности приложения
- Удобный интерфейс пользователя

##Требования для запуска
- .NET(от 8 и выше)
- Visual Studio 2022 или новее
- Открыть решение StudentManager.sln в Visual Studio
- Установить NuGet-пакеты(в том случае если их нет) - 1. Microsoft.Data.Sqlite.Core, 2.SQLitePCLRaw.bundle_e_sqlite3
- Запустить приложение
